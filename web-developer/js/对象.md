## Object.key

> JavaScript语言规定，对象的键名一律为字符串，所以，数组的键名其实也是字符串。之所以可以用数值读取，是因为非字符串的键名会被转为字符串。

>对象有两种读取成员的方法：“点”结构（object.key）和方括号结构（object[key]）。但是，对于数值的键名，不能使用点结构。

### Object.keys  遍历对象的key 键名
```js
var obj ={a: 1, b:2}
Object.keys(obj);    //["a", "b"]
```
